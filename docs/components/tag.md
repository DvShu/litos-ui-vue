# Tag 标签

通常用来展示一些属性

## 演示

<script setup>
  import { Tag } from "../../src"
</script>

### 基础用法

由 `type` 属性来选择 `tag` 的类型。也可以通过 `color` 属性来自定义背景色。

<ClientOnly>
  <CodePreview>
  <textarea lang="vue-html">
  <nt-tag>标签</nt-tag>
  <nt-tag class="ml-10" color="#409eff">标签</nt-tag>
  </textarea>
  </CodePreview>
</ClientOnly>
