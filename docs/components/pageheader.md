# PageHeader 页头

通常用于页面的顶部，显示标题以及返回按钮。常用于二级、三级页面。

## 演示

<script setup>
  import { PageHeader } from "../../src"
</script>

### 基础用法

基础用法

<ClientOnly>
  <CodePreview>
  <textarea lang="vue">

  </script>

  </template>
  </textarea>

    <PageHeader title="标题" sub-title="副标题"></PageHeader>
  </template>
  </CodePreview>
</ClientOnly>
