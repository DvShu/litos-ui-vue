2. 滚动条样式

import 'neatui-vue/style/util/scrollbar.css'">

使用

实际效果就跟文档的滚动条一样

主题定制 - 样式变量

组件提供了下列 CSS 变量，可用于自定义样式，

名称 描述 默认值
--nt-scrollbar-bg 滑块背景色 #ccc
--nt-scrollbar-width 滑块宽度 10px
--nt-scrollbar-hover 鼠标悬浮时滑块颜色 var(--nt-primary-color)
