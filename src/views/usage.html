依赖介绍

UI库除了依赖于 Vue3 外，还依赖于 ph-utils: 一个自己整理的一些常用的工具类

安装

npm i ph-utils @asteres/neatui-vue

按需引入

当前按需引入只支持在 Vite 中配合自动导入实现

安装依赖unplugin-vue-components + neatui-vue-resolver 

npm i unplugin-vue-components neatui-vue-resolver -D

Vite

在 vite.config.ts 的 plugins 添加如下代码： 

      import Components from 'unplugin-vue-components/vite'
      import NeatuiResolver from 'neatui-vue-resolver'

      plugins: [
        // ...
        Components({
          dts: 'src/components.d.ts',
          resolvers: [NeatuiResolver()],
        }),
      ]

浏览器兼容性

所有的组件尽量保证 ios 能够兼容到11
